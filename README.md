# Titanic Survival Analysis 🚢

## 📌 Project Overview
This project explores the Titanic dataset to uncover key factors that influenced passenger survival.

The goal is to perform Exploratory Data Analysis (EDA) and generate insights that can guide predictive modeling.

---

## 📊 Objectives
- Understand passenger demographics
- Identify survival patterns
- Analyze relationships between features and survival rate

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 🔍 Key Analysis Performed
- Data cleaning and handling missing values
- Univariate and bivariate analysis
- Feature relationships with survival
- Visualization of key trends

---

## 📈 Key Insights
- Females had a higher survival rate than males
- Passengers in higher classes had better survival chances
- Age and fare showed notable influence on survival
